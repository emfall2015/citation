# citation
##Description 

Ce projet est une application Python capable de récupérer automatiquement des citations depuis une API publique puis de générer une page HTML stylisée présentant ces citations sous forme visuelle.

---

##Fonctionnalites

L'application permet de découvrir :
-   les appels HTTP 
-   la manipulation de données JSON 
-   la génération automatique de contenu HTML 
-   une première approche de l'automatisation de production web.

---

##Technologies utilisees

-Python
-HTML5
-CSS3
-Git
